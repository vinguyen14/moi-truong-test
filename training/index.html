Kiến Trúc Dữ Liệu Cốt Lõi

Để hệ thống chạy mượt mà như mô phỏng trên, chúng ta sử dụng kiến trúc 3 lớp (3-Tier Architecture) hoàn toàn miễn phí của hệ sinh thái Google & GitHub:

1

Frontend (Giao diện)

Web HTML/Tailwind lưu trữ miễn phí trên GitHub Pages. Dùng hàm fetch() trong Javascript để bắn dữ liệu (Tên, Hình ảnh, Kịch bản) đi.

2

Backend API (Xử lý)

Viết bằng Google Apps Script. Đứng giữa bắt lấy dữ liệu từ Frontend gửi lên, sau đó phân loại xem đâu là text, đâu là file ảnh.

3

Database (Lưu trữ)

Text (Chữ) thì Backend sẽ ghi vào hàng cột trong Google Sheet. Nếu có File Ảnh/Video, Backend sẽ tự động tạo thư mục và đẩy lên Google Drive, rồi lấy link dán vào Sheet.

Bước 1: Chuẩn bị Đầu vào & Cấu hình AI

Làm quen Thuật ngữ hệ thống (Dành cho Non-IT)

Frontend Là "Mặt tiền" của căn nhà. Gồm giao diện, nút bấm, hình ảnh mà học viên nhìn thấy (Viết bằng HTML, Tailwind, JS).
Backend Là "Nhà kho & Não bộ". Xử lý logic, lưu trữ dữ liệu nộp bài của học viên. Chúng ta sẽ dùng Google Apps Script làm Backend.
API Là "Anh Shipper" giao tiếp giữa Frontend và Backend. Cầm bài thi của học viên chạy lên gửi cho Google Sheet.
Deploy Là hành động "Mở cửa đón khách". Đưa code của bạn lên mạng (Github Pages) để mọi người có link truy cập.

2Cấu hình Backend (Não bộ hệ thống)

Đây là đoạn mã Apps Script (code.gs) đã được cấu hình chuẩn xác nhất để xử lý dữ liệu và tự động tạo thư mục lưu ảnh. Học viên hãy copy toàn bộ đoạn mã này.

Thao tác: Tại màn hình Google Sheet -> Bấm Tiện ích > Apps Script -> Xóa mã mặc định đi -> Dán đoạn code bên dưới vào -> Tìm chữ YOUR_FOLDER_ID_HERE và thay bằng ID Thư mục bạn lấy ở Bước 1 -> Bấm Triển khai (Deploy) -> Tùy chọn Bất kỳ ai (Anyone) -> Copy Link API URL.
// ========================================== // BACKEND: TRAINER OS - KIOTVIET L&D // Cấu hình ID Thư mục gốc trên Google Drive // ========================================== const PARENT_FOLDER_ID = "YOUR_FOLDER_ID_HERE"; // <-- COPY ID THƯ MỤC CỦA BẠN DÁN VÀO ĐÂY const SHEET_NAME = "Submissions"; // Khắc phục lỗi CORS cho API function doOptions(e) { return ContentService.createTextOutput("") .setMimeType(ContentService.MimeType.TEXT) .setHeader("Access-Control-Allow-Origin", "*") .setHeader("Access-Control-Allow-Methods", "POST, GET, OPTIONS") .setHeader("Access-Control-Allow-Headers", "Content-Type"); } function doPost(e) { try { // Nếu không có data if (!e || !e.postData || !e.postData.contents) { return responseJSON({ success: false, message: "Không có dữ liệu gửi lên" }, 400); } // Phân tích Body JSON từ Frontend gửi lên const data = JSON.parse(e.postData.contents); const action = data.action; if (action === "submit") { return handleSubmit(data); } else if (action === "get_list") { return handleGetList(); } else if (action === "grade") { return handleGrade(data); } else { return responseJSON({ success: false, message: "Action không hợp lệ" }, 400); } } catch (error) { return responseJSON({ success: false, message: "Lỗi Server: " + error.toString() }, 500); } } // ------------------------------------------ // 1. NGHIỆP VỤ NỘP BÀI (HỌC VIÊN) // ------------------------------------------ function handleSubmit(data) { const studentName = data.student_name || "Khách Vô Danh"; const content = data.content || ""; const fileBase64 = data.file_base64 || ""; const fileName = data.file_name || "Minh_chung.png"; const mimeType = data.mime_type || "image/png"; let fileUrl = ""; // Nếu có file đính kèm, thực hiện lưu vào Drive if (fileBase64) { try { const parentFolder = DriveApp.getFolderById(PARENT_FOLDER_ID); // Kiểm tra xem Thư mục tên Học Viên đã có chưa let studentFolder; const folders = parentFolder.getFoldersByName(studentName); if (folders.hasNext()) { studentFolder = folders.next(); // Dùng thư mục cũ } else { studentFolder = parentFolder.createFolder(studentName); // Tạo thư mục mới } // Giải mã Base64 thành File const decodedData = Utilities.base64Decode(fileBase64); const blob = Utilities.newBlob(decodedData, mimeType, fileName); // Tạo file trong thư mục của Học viên và set quyền cho phép ai có link cũng xem được const newFile = studentFolder.createFile(blob); newFile.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW); fileUrl = newFile.getUrl(); } catch (e) { return responseJSON({ success: false, message: "Lỗi lưu file Drive: " + e.toString() }, 500); } } // Ghi dữ liệu vào Google Sheet const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME); if (!sheet) return responseJSON({ success: false, message: "Không tìm thấy tab Submissions trong Sheet" }, 500); // Cấu trúc mảng [Thời gian, Tên, Nội dung, Link Ảnh, Trạng thái] const rowData = [ new Date(), studentName, content, fileUrl, "Chờ chấm" ]; sheet.appendRow(rowData); return responseJSON({ success: true, message: "Nộp bài thành công!" }, 200); } // ------------------------------------------ // 2. NGHIỆP VỤ LẤY DANH SÁCH (GIẢNG VIÊN) // ------------------------------------------ function handleGetList() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME); if (!sheet) return responseJSON({ success: false, message: "Chưa cấu hình tab Submissions" }, 500); const data = sheet.getDataRange().getValues(); const results = []; // Bỏ qua dòng Header (dòng 1), lặp từ dòng 2 trở đi for (let i = 1; i < data.length; i++) { results.push({ row_index: i + 1, // Vị trí dòng thật trong Sheet timestamp: data[i][0], student_name: data[i][1], content: data[i][2], file_url: data[i][3], status: data[i][4] }); } // Đảo ngược mảng để bài nộp mới nhất lên đầu results.reverse(); return responseJSON({ success: true, data: results }, 200); } // ------------------------------------------ // 3. NGHIỆP VỤ DUYỆT BÀI (GIẢNG VIÊN) // ------------------------------------------ function handleGrade(data) { const rowIndex = data.row_index; if (!rowIndex) return responseJSON({ success: false, message: "Thiếu row_index" }, 400); const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME); // Cột E (Cột số 5) là trạng thái sheet.getRange(rowIndex, 5).setValue("Đã chấm"); return responseJSON({ success: true, message: "Chấm bài thành công!" }, 200); } // Hàm hỗ trợ xuất JSON có chứa Header CORS function responseJSON(obj, statusCode) { return ContentService.createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); }

3Thiết kế Web Frontend (Bằng AI)
