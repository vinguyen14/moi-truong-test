Bước 1: Hiểu Luật Chơi & Vẽ Bản Đồ

1. Nhận diện các loại "Ngôn ngữ"

Bạn không cần tự viết, nhưng khi AI nhả code ra, bạn phải biết nó thuộc loại nào để dán cho đúng.

HTML

Là BỘ KHUNG (tường, cột). Nó định hình xem web có mấy nút bấm, mấy ô nhập chữ.

CSS (Tailwind)

Là LỚP SƠN. Tạo màu sắc, hiệu ứng kính mờ, bo tròn góc cho giao diện đẹp mắt.

JS / Apps Script

Là ĐỘNG CƠ. Các đoạn code bắt đầu bằng chữ function. Giúp web tính toán, lưu dữ liệu.

JSON

Là GÓI HÀNG. Dữ liệu gói gọn để truyền mượt mà từ Web sang Google Sheet.

2. Trạm điều khiển AI Studio

Truy cập aistudio.google.com. Chọn mô hình Gemini 3.1 Pro Preview. (Dùng tài khoản Citigold để AI đủ thông minh đọc mã code dài ngàn dòng mà không quên luồng).

// Copy đoạn sau dán vào ô "System Instructions" để tạo kỷ luật cho AI: # ROLE & CONTEXT Bạn là Senior Fullstack Developer kiêm QA/Tester (HTML, Tailwind CSS, Vanilla JS, Google Apps Script). # 🚫 GOLDEN RULES 1. KHÔNG XÓA LOGIC/GIAO DIỆN CŨ. 2. BẢO TOÀN VĂN BẢN (Text tiếng Việt). 3. Xuất 100% code hoàn chỉnh. # ⚙️ ARCHITECTURE 1. API BẮT BUỘC dùng method 'POST', header 'text/plain'.

Tuyệt chiêu Prompt Giao diện (UI)

Khi nhờ AI làm web, hãy chèn thêm các câu thần chú sau để web nhìn xịn như hàng chục ngàn đô:

  • "Làm giao diện phong cách Glassmorphism (Kính mờ, trong suốt)."
  • "Dùng Google Font Be Vietnam Pro hoặc Nunito."
  • "Chừa sẵn thẻ <img> dạng tròn để tôi tự dán link Logo vào sau."

Bước 2: Dựng Database (Google Sheet)

1Tạo Bảng dữ liệu

Tạo file Google Sheet mới. Gõ các Tiêu đề cột (Ví dụ: Thời gian, Học viên, Nội dung...) ở dòng số 1.

2Gắn Code Backend

Vào Tiện ích > Apps Script. Xóa mã cũ, dán mã của AI vào. Bấm Ctrl + S để Lưu.

3Khởi chạy (Deploy) & Lấy Link API

Bấm Triển khai > Tùy chọn triển khai mới (chọn Web App, quyền Bất kỳ ai). Bạn sẽ nhận được 1 đường Link (URL).

Cực kỳ quan trọng: Copy Link API dán vào đâu?

Khi mang code Frontend về GitHub, bạn phải TÌM dòng const API_URL = "..." và dán đường link này vào giữa 2 dấu ngoặc kép. Nếu thiếu bước này, Web sẽ không bao giờ chạy!

Bước 3: Giao diện (GitHub)

Tạo Kho chứa (Repo)

Đăng nhập GitHub, bấm nút + -> New repository. Nhập tên dự án (VD: kiotviet-training).

Tạo File Code

Bấm Add file > Create new file. Tên file BẮT BUỘC kết thúc bằng .html hoặc .js (VD: index.html).

Thao tác Code (Chỉnh sửa / Lưu)

Sửa Code (Edit)

Bấm icon Cây bút hoặc trỏ chuột vào bảng code gõ phím E.

Lưu Code (Save)

Bấm nút xanh Commit changes HOẶC bấm nhanh Ctrl + S sau đó Enter.

Lấy Link Web (Đưa lên mạng)

Vào Settings > Pages. Chỗ Branch chọn Main -> Bấm Save. Chờ 1 phút tải lại trang sẽ thấy một đường Link. Đó chính là website hoàn chỉnh của bạn đem đi gửi cho Học viên!

Bước 4: Test & Sửa lỗi thông minh

Mẹo: Cắt Ghép Không Tốn Token

Đừng bắt AI viết lại cả file 1000 dòng. Vừa tốn Token vừa dễ sinh thêm lỗi mới. Yêu cầu AI chỉ xuất ra "Hàm bị lỗi". Sau đó làm theo cách sau:

Trên Github, bấm Ctrl + F. Gõ tên hàm (Ví dụ: function submit()). Bôi đen từ đầu đến cuối hàm đó và dán code mới đè lên.

Quy Tắc 2 Môi Trường (Test & Prod)

Tuyệt đối không sửa code trực tiếp trên Web đang có học viên dùng. Luôn tạo Repo du-an-test. Nhờ AI sửa -> dán vào Test -> Chạy mượt mà -> Mới Copy đè sang du-an-chinh.

Thực chiến: Build Hệ Thống

Copy các câu lệnh bên dưới đưa cho AI (Gemini), bạn sẽ tự tay làm ra được hệ thống y hệt như video mô phỏng ở Bài 1.

Lệnh tạo Backend (Mở AI Studio & Dán)

Chào AI, hãy viết mã Google Apps Script (hàm doPost/doGet). Bảng Google Sheet tên "Submissions", gồm cột: id, student, content, status. 1. action = 'submit': Lưu student và content, status là "Chờ chấm". 2. action = 'get_list': Đọc danh sách trả về JSON. 3. action = 'grade': Nhận ID, cập nhật status "Đã chấm". Trả code hoàn chỉnh, dùng try-catch.

Lệnh tạo Frontend (Giao diện Web)

Bây giờ viết 2 file HTML dùng Tailwind CSS: 1. student.html: Form nộp bài đẹp (Glassmorphism, font Nunito). Nút bấm gọi Fetch API gửi action='submit'. 2. teacher.html: Dashboard quản trị tông màu tối. Fetch API action='get_list' để hiển thị Bảng. Có nút "Chấm" gọi action='grade'. Code để sẵn 1 biến const API_URL = "" ở đầu script để tôi tự điền link.
