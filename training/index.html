Bước 1: Phân tích Insight & Setup Trạm AI

1. Phân tích Tính năng (User Insight)

Để xây hệ thống SaaS, bạn phải biết rõ Học viên cần gì, Giảng viên cần gì để lát nữa ra lệnh cho AI viết code cho chuẩn.

Insight Học Viên (Frontend)

  • Cần popup khai báo Tên/Phòng ban.
  • Giao diện làm bài Form nhiều ô.
  • Có nút Nộp Bài + Hiệu ứng Loading.
  • Có thông báo "Nhận xét mới" màu đỏ.

Insight Giảng Viên (Admin)

  • Thấy Bảng danh sách bài tập.
  • Bộ lọc tìm kiếm theo Trạng thái.
  • Biết bài nào là "Bổ sung sửa lại".
  • Khung popup để Nhập điểm, phê duyệt.

2. Chuẩn bị Tài nguyên "Gốc rễ"

Đừng bắt AI viết web từ con số 0. Hãy ném file dự án cũ (lưu trên Google Docs) để AI hiểu "Gốc rễ" phong cách code của bạn. Bạn copy mã code cũ lưu vào Docs như hình dưới.
